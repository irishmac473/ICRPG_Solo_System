# ICRPG Solo System

A lightweight **solo play framework** for [Index Card RPG](https://www.runehammer.online/index-card-rpg).

This project adds a small set of **house rules, procedures, and tables** that allow ICRPG to be played without a GM.

The goal is to keep the system **fast, procedural, and interpretation-driven**, using existing tools whenever possible.

—

## Required Tools

This system assumes access to the following tools:

- [Index Card RPG (ICRPG)](https://www.runehammer.online/index-card-rpg)
- [Gamemaster’s Apprentice Deck 2e](https://www.drivethrurpg.com/en/product/475920/the-gamemaster-s-apprentice-2e-base-deck)
- [PUM Companion (PUMc)](https://unfolding-machines.com)

Optional:

- [ICRPG Target Cards](https://www.drivethrurpg.com/en/product/290540/icrpg-target-cards-pl-en)

—

## Repository Structure

Each genre contains three documents:

**House Rules**

Modifications to the base ICRPG rules.

Examples:
- Level 0 characters
- equipment and load system
- hero coin tweaks
- carousing

**Solo Procedures**

Step-by-step procedures for running the game solo.

Examples:
- scene procedure
- travel procedure
- delving procedure
- NPC generation

**Tables**

All random tables used by the system.

Tables are formatted for easy entry into **PUM Companion**.
