# SCENE PROCEDURE

Scenes represent encounters, conversations, delving rooms, new hexes, or major events.

**Scene procedure**
Draw TARGET card
Draw GMA Room Card
Interpret environment
Search for traps
Draw GMA Oracle Card
Resolve scene

# DELVING PROCEDURE

Each room in a dungeon or ruin is treated as a scene.

## Enter Room

Draw TARGET card.

Draw **GMA Card**.

Determine the room.

Room size

Width = d12 result × 5 ft
Length = d8 result × 5 ft

Room exits

Use the compass icon on the card.

Room theme

Interpret the card illustration.

Room details

Use the sensory prompt line.

## Trap Check

Immediately search for traps.

Roll DEX vs TARGET.

Draw **GMA Oracle Card**.

Ask

Are there traps?

If NO

Continue room normally.

If YES

Check the search roll.

Search success

Trap detected.

You may disarm, avoid, or bypass it.

Search failure

Trap triggers.

Roll DEX vs TARGET to dive clear.

Success → avoid trap or take half damage
Failure → trap hits normally

Trap severity based on TARGET

TARGET 10–11 → minor trap
TARGET 12–13 → dangerous trap
TARGET 14–15 → deadly trap
TARGET 16+ → brutal trap

## Hidden Features

If the room suggests something hidden

Roll vs TARGET.

Success → feature discovered
Failure → feature permanently missed

Only one attempt may be made.

Hidden reward quality is based on TARGET.

## Room Content

Use GMA keywords and imagery to determine

monsters
NPCs
hazards
clues

Resolve encounters using normal ICRPG rules.

## Dungeon Progression

After resolving the room roll on **Dungeon Progression Table**.

# TRAVEL PROCEDURE

Travel occurs in daily cycles.

Each day contains four travel segments.

Morning
Midday
Afternoon
Evening

Movement

1 hex per segment
2 hex per segment when traveling by road

Travel procedure

Determine weather
Navigate
Scout terrain
Check landmarks
Resolve encounters
Continue travel

# WEATHER PROCEDURE

Determine starting weather using seasonal weather table.

Each day roll on **Weather Change Table**.

Weather may

Remain the same
Worsen one step
Improve one step
Shift dramatically

# NAVIGATION

Choose the PC with the highest INT or WIS.

Roll against TARGET.

Failure results determined on **Navigation Failure Table**.

If lost

Travel continues but direction is unknown until navigation succeeds.

# SCOUTING

A character may scout from elevated terrain.

Roll INT or WIS.

Success

Reveal terrain using **Terrain Visibility Table**.

Reveal visible roads and rivers.

Perform **Landmark Check**.

Failure

Roll on **Scouting Failure Table**.

# TERRAIN GENERATION

When entering unexplored hex

Roll on **Terrain Region Table**.

Future hexes follow the matching **Terrain Continuity Table**.

# LANDMARK CHECK

Landmarks may be discovered during travel or scouting.

Roll on **Landmark Table**.

Landmarks may introduce new locations or story threads.

# CAMPING

Determine area danger level.

Roll on appropriate table

Safe Camp Table
Borderland Camp Table
Dangerous Camp Table

# HUNTING AND FORAGING

Consumes one travel segment.

Foraging

Roll WIS.

Resolve encounter using **Foraging Encounter Table**.

Hunting

Roll DEX or WIS.

Roll on **Game Table**.

Apply terrain and seasonal modifiers.
